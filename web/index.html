O Conceito

Living System & Minimalismo

O repositório foi reimaginado como uma galeria de arte brutalista com forte influência dark mode e tons de cinza puro.

The Playful Serious

Interações sutis transformam a navegação no catálogo. Cada componente possui micro-animações (hover magnético e glows suaves) para engajar a exploração.

The Unexpected Medium

Em vez de um painel aborrecido de SaaS, a UI serve como um laboratório visual interativo. Utilize filtros que aplicam "lentes" e isolam os elementos como obra principal.

Colaboração Git Base

Importe ou exporte JSON estruturado. Abra PRs em data/catalog.json para integrar curadorias ao sistema unificado.

Glassmorphism Ativo

Filtros dinâmicos e renders em iframe beneficiam-se de pesados efeitos de blur e overlays prateados, dando peso e luxo a paletas de cores simples.
